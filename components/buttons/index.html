Buttons

Buttons come in three varieties: download buttons, standard buttons and user flow buttons.

Download buttons

download

HTML


<a href="#" class="button-green">download</a>

Standard buttons

HTML


<button class="button">default</button>
<button class="button insensitive">insensitive</button>

User flow buttons

HTML


<button class="button">next</button>
<button class="button-negative">no</button>
